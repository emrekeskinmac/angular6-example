Proje Hakkında

Projede kayıt olup giriş yapılabilir.
Kendo ui ile chart ve grid componentlerini kullanarak bir dashboard alanı yaptım.
Menü bölümünü yaparken bootstrap classlarını kullanarak yaptım.
Iconlarda font-awesome kullandım.
Kullanıcıları localstroge kullandım ve buna 2 adet servis yazdım.

Proje Komutları

ng serve kod ile projeyi çalıştırabilirsiniz ng build kod ile prod build alabilirsiniz ng test kod ile projeyi test edebilirsiniz ng lint kod ile projenin lint testi yapabilirsiniz ng e2e kod ile e2e testi çalıştırabilirisiniz

CSS (SCSS) olarak yapılanlar

z-widget olarak dashboard scss ile bi BEM css metodolojisi kullanarak yazdım
Tabi burda oocss veya smacss metodundan herhangi birini projeye kullanılabilir.

TS (Typescript) olarak yapılanlar

TSlint özelliği kullanılarak tüm kod kurallarına göre yazılmıştır
2 tane servis yazdım. User ve Auth servisini yazdım.
facebackend.ts olarak HttpRequest hooklayıp ona gelen isteği direk localstroge daki veriyi kullanılarak yazdım.
ullanıcı doğrulamasını jwt olarak kendimin verdiğim bir token ile yaptım
Layoutlara göre doğrulama yapılarak girilmesini sağlandı. 1 tane guard ile bunu kontrol ettim
Serviste kullanılan modelleri models olarak yazdım.
